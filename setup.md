
# Install instructions

- Contains instructions for seting up Fairlearn, and an example
https://fairlearn.org/main/quickstart.html

Main instructions:
pip install fairlearn

https://pypi.org/project/raiwidgets/

Main instructions:

pip install raiwidgets
